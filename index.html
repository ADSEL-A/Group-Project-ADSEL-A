About the project

The project that we will be working in this unit is Data Science Visualisations for Education for Australia Data Science Education Institute (ADSEI). The problem this project aims to solve is to help students understand and explore the data of the realworld datasets is an easy and effective manner, as the ones with data skills will have an advantage as this field develops. Since, in today’s time we don’t have a lot of teachers with data and technical skills who could teach the students about data, we need to develop a tool that will allow teachers and their students to explore datasets in an interactive and engaging way without needing serious technical skills. Hence, this project aims to solve the problem in Education domain.
